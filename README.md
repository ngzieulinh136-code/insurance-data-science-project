## Project Description
This project focuses on cleaning, analyzing, and visualizing health insurance data to identify the factors that most strongly affect medical insurance charges.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis
- Insight Extraction
